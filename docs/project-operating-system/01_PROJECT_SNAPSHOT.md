# Project Snapshot

## App Name

The Agency / Agency Agents

## App Type

- AI agent catalog
- Local web app
- Static browser tool
- Registry/import/export tooling
- Project orchestration toolkit
- Documentation and strategy system

## What This Is

A curated collection of specialist AI agent prompts, workflows, and strategy playbooks, plus tools for browsing, matching, importing, normalizing, and exporting agent packs.

## What It Does

- Stores specialist agent definitions as Markdown content.
- Builds a searchable local/static agent matchmaker catalog.
- Supports optional Gemini-powered matching through a local web server.
- Provides multi-source registry foundations for importing and normalizing external agent/skill packs.
- Provides NEXUS strategy docs for orchestrating agents across project phases.
- Includes an experimental `supabase-web` app for future web-facing work.

## Who It Serves

- Builders who want reusable AI specialists.
- Developers who want local agent matching and exportable agent packs.
- AI-assisted teams that need structured project workflows.
- Maintainers expanding this repo without losing existing content.

## Main Outcome

A user can discover the right agent or agent workflow, inspect the source content, and use/export it in their preferred tool without damaging the curated catalog.

## Current Status

Improving. The repo has working content and tooling, plus active uncommitted development around the registry, matchmaker, scripts, and `supabase-web`.

## One-Line Summary

Agency Agents is an AI specialist catalog and matchmaker with registry/export tooling and project orchestration docs.
