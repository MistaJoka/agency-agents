# Features

## Core Feature

Curated AI agent discovery and usage through Markdown source content and a local/static matchmaker.

## MVP Features

- Curated agent Markdown catalog.
- Static matchmaker with search/filter/offline heuristic ranking.
- Local web server with optional Gemini matching.
- Catalog build script.
- Multi-source registry foundation with opt-in source sync.
- Registry export commands.
- NEXUS strategy and handoff docs.

## Supporting Features

- Tool-specific integration outputs.
- QA reality checks for the matchmaker.
- Workflow docs under `docs/workflows/`.
- Project operating-system docs in this folder.

## Later Features

- Registry-backed matchmaker UI.
- Hosted web experience.
- Better source provenance UI.
- Pack publishing workflows.
- Supabase-backed features if confirmed.

## Rejected for Now

- Replacing the repo with generic template folders.
- Mandatory authentication.
- Mandatory cloud database.
- Auto-fetching remote sources by default.

## Feature Rule

If a feature does not improve agent discovery, source quality, import/export reliability, or project orchestration, park it.
